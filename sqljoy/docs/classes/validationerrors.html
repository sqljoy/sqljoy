Class ValidationErrors

ValidationErrors accumulates validation errors in the errors property.

It's designed to support showing validation errors for user input.

errors

errors: Record<string, string>
  • Defined in projects/flowstate/flowstate/sqljoy-client/src/validation.ts:48

An object mapping the invalid parameter name to an error. If there is more than one error per parameter, only the first is recorded.

If an error doesn't pertain to any individual parameter, it's stored in the nonFieldErrors array.
