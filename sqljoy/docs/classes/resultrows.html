Class ResultRows

ResultRows represents an iterable query result over the result rows as Row objects.

If there are no rows in the result, the iterable is empty. A server function/transaction can return arbitrary JSON including one or more Results.

remarks

PostgreSQL doesn't have a client function to get the last inserted id. It's recommended you use RETURNING id on your INSERT query, and the resulting Result object will contain a single Row of {id: number}.
